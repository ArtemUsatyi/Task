# Java Examples Project

Этот репозиторий содержит коллекцию примеров кода на Java, демонстрирующих различные концепции программирования.

## Структура проекта

```
src/
├── BankingSystem/          # Система банковских операций
├── Booking/                # Система бронирования отелей
├── Collection/             # Примеры работы с коллекциями
├── Polymorphism/           # Примеры полиморфизма
├── constructor/            # Примеры конструкторов
├── com/                    # Дополнительные пакеты
├── Example01.java - Example11.java  # Базовые примеры
├── Main.java               # Главный файл запуска
├── SimpleNumber.java       # Пример работы с числами
└── SodaDrink.java          # Пример класса напитка
```

## Содержание

### Основные примеры
- **Example01-11.java** - Набор базовых примеров, демонстрирующих фундаментальные концепции Java
- **Main.java** - Точка входа для запуска проекта

### Модули

#### BankingSystem
Система управления банковскими счетами и операциями:
- Account - банковский счёт
- Bill - счета и платежи
- Deposit - депозиты
- Person - клиент банка
- Service/DepService - сервисные классы

#### Booking
Система бронирования отелей:
- Hotel - отель
- Room - номер
- Client - клиент
- BookingService - сервис бронирования
- Bill - счета

#### Collection
Примеры работы с коллекциями Java:
- ArrayListNames - работа с ArrayList
- TelephoneDirectory - телефонный справочник
- ArrayVSLinked - сравнение массивов и связных списков

#### Polymorphism
Примеры использования полиморфизма в Java

#### Constructor
Примеры использования конструкторов классов

## Требования

- Java Development Kit (JDK) 8 или выше
- Любая IDE для Java (IntelliJ IDEA, Eclipse, NetBeans)

## Компиляция и запуск

### Компиляция из командной строки
```bash
cd src
javac Main.java
```

### Запуск
```bash
java Main
```

### Использование IDE
1. Откройте проект в вашей IDE
2. Дождитесь индексации проекта
3. Запустите Main.java
